# 3.2 触发时机

## 1. 触发时机概述

触发时机是技能发动的关键时间点，主要分为：
- [阶段类触发](#阶段类)
- [事件类触发](#事件类)
  - [状态类触发](#状态类)
  - [特殊类触发](#特殊类)
- [全局类触发](#全局类)

## 2. 阶段类触发<a id="阶段类"></a>

### 2.1 角色阶段
```javascript
trigger: {
    player: "enterGame",     // 进入游戏时
    player: "phaseBefore/Begin/End/After",   // 回合开始前/开始时/结束时/结束后
    player: "phaseZhunbei",  // 准备阶段
    player: "phaseJudge",    // 判定阶段
    player: "phaseDraw",     // 摸牌阶段
    player: "phaseUse",      // 出牌阶段
    player: "phaseDiscard",  // 弃牌阶段
    player: "phaseJieshu",   // 结束阶段
}
```
#### 阶段顺序：
- enterGame
- phaseBefore
- phaseBeforeStart
- phaseBeforeEnd
- phaseBeginStart
- phaseBegin
- phaseZhunbei
- phaseJudge
- phaseDraw
- phaseUse
- phaseDiscard
- phaseJieshu
- phaseEnd
- phaseAfter

### 2.2 阶段细分
```javascript
trigger: {
    player: "phaseDrawBegin",     // 摸牌阶段开始时
    player: "phaseDrawEnd",       // 摸牌阶段结束时
    player: "phaseUseBegin",      // 出牌阶段开始时
    player: "phaseUseEnd",        // 出牌阶段结束时
    player: "phaseDiscardBegin",  // 弃牌阶段开始时
    player: "phaseDiscardEnd",    // 弃牌阶段结束时
}
```

## 4. 全局触发<a id="全局类"></a>

### 4.1 基本用法
```javascript
trigger: {
    global: "gameStart",      // 游戏开始时
    global: "roundStart",     // 轮数开始时
    global: "phaseBegin",     // 任意角色回合开始时
    global: "damageEnd",      // 任意角色受到伤害后
    global: "washCard",       // 洗牌时
}
```

### 4.2 多重触发
```javascript
trigger: {
    player: ["phaseBegin", "phaseEnd"],     // 回合开始和结束时
    global: ["damageEnd", "loseHpEnd"],     // 任意角色受到伤害或失去体力后
}
```

## 5. 触发优先级

### 5.1 优先级设置
```javascript
"priority_skill": {
    trigger: {player: 'phaseBegin'},
    priority: 5,              // 设置优先级(默认为1)
    forced: true,
    content: function(){}
}
```

### 5.2 常见优先级
- 10: 最高优先级
- 5: 高优先级（无懈可击）
- 1: 默认优先级
- -1: 低优先级
- -5: 最低优先级

## 6. 触发条件

### 6.1 基本判断
```javascript
filter: function(event, player){
    // 血量条件
    return player.hp < 3;
    
    // 手牌条件
    return player.countCards('h') > 0;
    
    // 目标条件
    return event.player != player;
    
    // 牌名条件
    return event.card && event.card.name == 'sha';
}
```

### 6.2 复杂条件
```javascript
filter: function(event, player){
    // 多重条件
    if(player.hp < 3 && player.countCards('h') > 0){
        return event.player.isAlive() && 
               event.player.countCards('h') > 2;
    }
    return false;
}
```

## 7. 进阶用法

### 7.1 触发顺序控制
```javascript
"order_skill": {
    trigger: {player: 'phaseBegin'},
    content: function(){
        // 打断后续触发
        trigger.cancel();
        
        // 自定义触发
        event.trigger("order_skill")
        
        // 跳过特定阶段
        player.skip('phaseDraw');
    }
}
```

### 7.2 条件触发
```javascript
"condition_skill": {
    trigger: {player: 'phaseBegin'},
    direct: true,             // 锁定技且不输出日志
    check: function(event, player){
        return player.hp < 3; // AI发动条件
    },
    content: async function(){
        let target = await player.chooseTarget('对一名角色造成伤害，然后你失去一点体力').forResult();
        if (target.bool){
            player.logSkill("condition_skill",target.targets[0])
            await target.targets[0].damage()
            await player.loseHp()
        }
    }
}
```

1. 创建一个多重触发技能：
   - 在回合开始和结束时触发
   - 根据不同时机有不同效果
   - 添加触发条件

<details>
<summary>参考答案 | 🟩 Easy</summary>

```javascript
"multi_trigger": {
    // 多个触发时机
    trigger: {
        player: ["phaseBegin", "phaseEnd"]
    },
    // 触发条件
    filter: function(event, player){
        if(event.name == 'phaseBegin'){
            return player.countCards('h') < 3; // 回合开始时手牌少于3
        }
        return player.hp < 3; // 回合结束时体力值少于3
    },
    // 根据时机执行不同效果
    content: async function(){
        if(trigger.name == 'phaseBegin'){
            // 回合开始时摸牌
            await player.draw(2);
            game.log(player, '触发了回合开始效果');
        } else {
            // 回合结束时回血
            await player.recover();
            game.log(player, '触发了回合结束效果');
        }
    },
    ai: {
        threaten: 1.5
    }
}
```

2. 创建一个全局触发技能：
   - 监听所有角色的特定事件
   - 根据条件决定是否触发
   - 实现连锁效果

<details>
<summary>参考答案 | 🟨 Medium</summary>

```javascript
"global_trigger": {
    // 监听所有角色的伤害事件
    trigger: {
        global: "damageEnd"
    },
    // 触发条件
    filter: function(event, player){
        return event.player != player && // 不是自己受伤
               event.player.isAlive() && // 目标存活
               event.num > 0; // 伤害大于0
    },
    // 连锁效果
    content: async function(){
        // 记录伤害数值
        event.num = trigger.num;
        // 选择效果
        let choices = ['摸牌', '回血'];
        if(player.countCards('h') > 0) choices.push('弃牌造成伤害');
        
        let result = await player.chooseControl(choices)
            .set('prompt', '请选择一个效果')
            .set('ai', function(){
                if(player.hp <= 2) return '回血';
                if(player.countCards('h') < 2) return '摸牌';
                return '弃牌造成伤害';
            })
            .forResult();
        // 执行效果
        switch(result.control){
            case '摸牌':
                await player.draw(event.num);
                break;
            case '回血':
                await player.recover(event.num);
                break;
            case '弃牌造成伤害':
                await player.chooseToDiscard(1, true);
                let target = await player.chooseTarget('选择一名角色造成伤害').forResult();
                if(target.bool){
                    await target.targets[0].damage();
                }
                break;
        }
    },
    ai: {
        threaten: 2,
        expose: 0.2
    }
}
```
</details>

3. 创建一个优先级技能：
   - 设置高优先级
   - 控制其他技能的触发
   - 实现特殊效果

<details>
<summary>参考答案 | 🟨 Medium</summary>

```javascript
"priority_skill": {
    // 设置高优先级
    priority: 10,
    // 触发时机
    trigger: {
        target: "useCardToTargeted"
    },
    forced: true,
    // 触发条件
    filter: function(event, player){
        return event.card.name == 'sha' && // 目标是【杀】
               player.countCards('h') > 0;  // 有手牌
    },
    // 特殊效果
    content: async function(){
        // 取消原有事件
        trigger.cancel();
        game.log(player, '触发了优先级技能');
        // 展示一张手牌
        let card = await player.chooseCard('h', true, '请展示一张手牌').forResult();
        if(card.bool){
            await player.showCards(card.cards);
            
            // 根据花色执行效果
            if(get.color(card.cards[0]) == 'red'){
                await player.draw();
                game.log(player, '展示红色牌，摸一张牌');
            } else {
                await trigger.player.draw();
                game.log(trigger.player, '展示黑色牌，使用者摸一张牌');
            }
        }
    },
    ai: {
        effect: {
            target: function(card, player, target){
                if(card.name == 'sha') return 0.5;
            }
        }
    }
}
```
</details>
